Command Line Basics

دليلك الشامل لأساسيات سطر الأوامر. يجمع المشروع أهم الأوامر لأنظمة Linux و Windows CMD و PowerShell في مكان واحد، مع مراجع احترافية للوصول السريع.

Linux

استكشف قوة نظام لينكس مع أوامر إدارة النظام، الملفات، الشبكات، والصلاحيات.

Windows CMD

أتقن موجه الأوامر التقليدي في ويندوز مع أدوات إدارة الملفات، الشبكات، وأدوات التشخيص.

PowerShell

انتقل إلى مستوى متقدم من الأتمتة وإدارة الأنظمة مع أوامر PowerShell الحديثة.
