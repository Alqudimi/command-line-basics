⚙️ Command Line Basics

مرجع شامل يجمع أهم الأوامر الأساسية لأنظمة لينكس، CMD، وPowerShell في مكان واحد، مع مراجع منظمة للوصول السريع.

Linux

إدارة النظام، التعامل مع الملفات، والتحكم بالشبكات والصلاحيات.

Windows CMD

أوامر إدارة الملفات، أدوات التشخيص، والتحكم في نظام ويندوز التقليدي.

PowerShell

الأتمتة الحديثة، إدارة الأنظمة المتقدمة، وكتابة السكربتات القوية.
